# Локальный запуск микросервисов:
1. Переходим в папку с микросервисом
2. Прописываем команду uvicorn main:app --reload
3. Переходим по ссылке, которая вернулась в терминале

# Запуск из контейнера
1. Переходим в папку с микросервисом
2. Прописываем команду docker build -t fastapi-app .
3. После того, как контейнер соберется, мы можем запускать приложение прямо из контейнера командой docker run -p fastapi-app
4. Переходим в браузере по ссылке http://localhost:8000(8001)/docs - хост зависит от микросервиса
